# Chronovox

Première interface expérimentale de navigation temporelle.
Cette version couvre uniquement le sens **futur**. Le passé est en cours de calibration (voir [Feuille de route](#feuille-de-route)).

> ⚠️ Aucun paradoxe temporel n'a été créé pendant le développement de ce projet. Probablement.

## Aperçu

Chronovox est une console de translation temporelle qui permet d'entrer une année cible, d'engager le flux, et de recevoir un rapport d'observation en provenance du futur. Les résultats sont scientifiquement invérifiables.

## Fonctionnalités

- 🎯 Choisir une année cible (2026 et au-delà)
- 🌀 Stabiliser le flux temporel avant translation
- ⚡ Détecter les paradoxes en temps réel
- 🚫 Refuser tout voyage avant 2026 (protocole de sécurité non négociable)
- 📡 Générer un rapport d'observation aléatoire depuis le futur

## Installation

Aucune installation requise. Ouvrir `index.html` dans un navigateur.

```bash
open index.html
```

## Utilisation

1. Entrer une année dans le champ prévu à cet effet
2. Cliquer sur **Engager**
3. Ne pas éteindre l'ordinateur pendant la stabilisation du flux
4. Lire le rapport d'observation

## Limitations connues

- Les dinosaures ne répondent pas aux tentatives de contact
- Les années négatives provoquent un paradoxe non résolu
- Einstein n'a toujours pas accepté la pull request
- Le voyage dans le futur ne permet pas de connaître les numéros du loto (testé, ça ne marche pas)
- Un seul utilisateur à la fois peut occuper la ligne temporelle
